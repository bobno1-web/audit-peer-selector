### 실행 불가로 판명된 항목 + 대안
1. **비상장 재무제표 (구조화 API) — 불가 (0%).**
   대안: Loop 0-D의 수집 경로를 `fnlttSinglAcntAll` → **감사보고서 XBRL/원본파일 파싱**으로 변경.
   근거: 데이터는 감사보고서 문서에 존재(표본 38% 제출). 단, 파싱 난이도·커버리지는 별도 스파이크로 확인 필요.
2. **비상장 사업내용 텍스트 — 불가 (사업보고서 0%).**
   대안: (a) 비상장은 `features/business` 피처를 비워 두고 산업분류·규모만으로 매칭, 또는
   (b) 감사보고서 내 텍스트 활용 가능성 조사(미확인).
3. **시점별 산업분류 스냅샷 — API 없음 (현재값만).**
   대안: 과거 보고서에서 시점 산업 유도. 현재값 그대로 쓰면 **룩어헤드 위반**이므로 현재값 사용은 금지.
4. **상장폐지 시점 — corpCode로 불가.**
   대안: KRX 상장폐지 목록 등 외부 소스 확보(생존편향 방지에 필수).
