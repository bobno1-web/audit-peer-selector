# LOOP_WEB4 — 결과 화면 정리 + 브랜드 마크 적용

## 이번 루프 (표시 계층만, 엔진·데이터 무변경)
A) 결과 화면 실무용 정리 — 순위 숫자 화면 표시 제거 + 규모그룹 접이식
B) 6개 자매 프로젝트 공통 브랜드 마크(돋보기) 적용 — 브랜드바 + favicon
★ 엔진·채점기·데이터·라우팅·산출물·UI색 무변경. 표시·로고만. 정답지·test 파일 미열람.

## PART A. 결과 화면 정리
- **A-1 순위 숫자 제거**: peer 항목의 순위 숫자(1,2,3…) **화면 표시만** 제거. 엔진 순위 데이터·
  내부 순서(유사도 내림차순)는 유지. 유사도 점수는 우측에 그대로.
- **A-2 그룹 구조**(Web-3 top_axes 분류 유지, 표시만 변경):
  - 섹션1(메인, 항상 펼침) "비교 가능한 동종기업" = 업종/사업내용 유사 peer.
  - 섹션2(접이식 `<details>`, 기본 닫힘) "업종은 다르나 규모·시가총액이 유사한 기업" = 규모/시총만
    유사(업종·사업내용 없음). 닫힌 상태에서도 "N곳" 표시.
- **A-3 업종유사 0**: 섹션1을 "이 회사와 업종이 비슷한 동종기업을 찾지 못했습니다. 규모가 큰
  기업은 비교 대상이 마땅치 않을 수 있습니다." 안내로. 규모·시총 유사는 섹션2(접이식)로 계속 제공.
  억지로 섹션1을 채우지 않는다.
- **A-4 왜곡 금지**: 유사도·회사명·태그는 실제 출력 그대로. 분류는 기존 top_axes 만(새 로직 0).

## PART B. 브랜드 마크 (6개 자매 공통 규격)
- **B-1 규격**: 둥근 사각 타일(radius ≈ 타일 1/4) + 흰/연회색(#F9FAFB) 배경 + 얇은 회색 테두리
  (#E5E7EB 1px), 심볼 = 블랙 단색(#1A1A1A) 돋보기(렌즈 원 + 중앙 점 + 대각 손잡이). 인라인 SVG.
- **B-2 적용**: 브랜드바 제품명 왼쪽 마크(기존 코랄/도트 아이콘 교체) + favicon 동일 심볼(data URI SVG).
- **B-3 규율**: UI 색(코랄) 무변경(마크는 블랙 단색). 제품명·기능·텍스트·파이프라인 무변경.
