# PLAN — 합의된 계획 (기준 문서)

이 파일이 기준이다. 이후 나와 사용자가 딴소리하면 이 파일을 따른다.
바꾸려면 `docs/DECISIONS.md`에 남긴다.

## 제품 정의
- **입력**: 감사 대상 기업 1개 (상장/비상장).
- **출력**: 비교 가능한 동종기업 목록 (기본 5개, 순위 포함).
- **비포함**: peer와의 비교/분석 기능. 사용자가 이미 보유한 툴이 담당한다.
  우리는 **목록만** 뱉는다.

## 확정 사항
- **목적**: 초도감사 분석적 절차용 동종기업 선정.
- **유니버스**: **KOSPI + KOSDAQ (상장사만)**. ~~비상장 외감법인~~ → **NO-GO 최종 확정(D-008, Loop 0-E)**.
  - **구조적 한계(명시)**: 비상장 감사대상의 peer는 전부 상장사이므로 규모가 더 크다.
    이는 이 도구의 구조적 한계이며 **사용자가 인지하고 써야 한다.**
  - 근거(0-E 재판정, 종업원수 오류 철회 후 공정 재시험): 필수 6계정 **전부 확보** 성공률 **51%(<70%)**,
    결측이 부동산·건설에 쏠림 (`docs/DECISIONS.md` D-008, `docs/SPIKE_UNLISTED.md` 갱신 전 0-D 근거).
  - **생존편향 — Loop 0-F에서 해결**: KRX 상폐일이 없어도 **공시이력(사업보고서 제출)**으로
    시점별 유니버스를 복원한다(`docs/SURVIVORSHIP.md`). 상폐 표본 40/40이 폐지 후 명단에서
    정확히 빠짐(+최근 재무 없음 확증), 퇴출 489건 실측. 게이트는 대리지표(규모 단조성)로 문자상
    FAIL이나 직접증거로 방식 안전 입증(D-009, 번복가능). 이전의 "KRX 필요" 한계는 대체됨.
- **제외**: 스팩, 리츠, 우선주, ETF (사업 실체 없음). 판정은 **구조적 근거**(induty_code·상장주식종류)로 하며 종목명 키워드 금지.
- **관리종목·상장폐지기업**: T 시점에 존재했다면 포함 (생존편향 금지).
- **기간**: 2015~2025.
  - 개발용(dev): 2015~2022 / 봉인(holdout): 2023~2025.
- **평가시점**: 매년 5월 15일 (12월 결산 보고서 제출 후).
- **미국/해외**: **[확정 — 영구 제외]**.
- **출력 포맷**: 미정. 사용자 기존 툴 스펙 확인 후 결정. `[TODO: 결정필요]`.

## 루프 계약
- 0-A 골격 (완료)
- 0-B 문서/규칙/훅 (완료)
- 0-C 계획확정 + 타당성조사 (완료)
- 0-D 하네스 수리 + 비상장 NO-GO (완료)
- 0-E 비상장 최종 재판정 + 유니버스 정제 (완료)
- 0-F 하네스 반영 + 생존편향 해결 + PIT 데이터 구축 (완료) ← **Loop 0 종료**
- 1  baseline 엔진 → 기준 점수 확보 ← 다음
- 2  similarity 엔진 → baseline과 결판
- 3+ 개선 (내용은 Loop 2 결과가 정한다. 미리 정하지 않는다)
- 최종 봉인 개봉 (2023~2025). **딱 한 번.**

## 승리 조건 (미리 못 박는다)
similarity 엔진이 baseline 대비 **오차 중앙값을 20% 이상 낮춰야** 한다.
> 예: baseline 28% → **22.4% 이하**.

그 미만은 "이겼다"고 하지 않는다. **결과를 보고 이 기준을 바꾸지 않는다.**

## 종료 조건
1. holdout에서도 baseline을 이긴다.
2. 임의 상수가 코드에 **0개**.
3. peer 목록이 사용자 기존 툴에 들어간다.

셋 다 충족하면 종료. 하나라도 안 되면 계속.
