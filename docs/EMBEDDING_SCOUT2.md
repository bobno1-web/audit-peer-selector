# EMBEDDING_SCOUT2 — 신경망 임베딩 무료 정찰 (Loop 6 PART 3)

유료 최신 임베딩에 돈 쓸 가치가 있는지, **무료 소형 신경망 임베딩**으로 실측 정찰한다.
Loop 4 정찰(TF-IDF text-off 실험)은 "텍스트 표현 정교화의 상한"을 간접 시사했다. 이번엔 **실제
신경망 의미 임베딩**을 돌려 TF-IDF(문자 3-gram)와 직접 비교한다.

- 모델: **`minishlab/potion-multilingual-128M`** (model2vec 정적 임베딩, dim=256). 로컬 pip 설치,
  **torch 불요**, 유료 API 0. 로컬 캐시(`data/pit/cache/m2v_multilingual`, 512MB) — 재실행 재다운로드 0.
- 방법: `scripts/loop6_embedding_scout.py`. **같은 텍스트**(section_text.parquet 2,752건 'ok') ·
  **같은 격자**(5^5=2,851 조합) · **같은 SEED=20260715·SUB=250** · 같은 채점기·페널티(2.0263).
  **유일한 변수 = 텍스트 벡터라이저**(문자 3-gram TF-IDF vs 신경망 의미 임베딩). L2정규화 코사인.
- ★ 캐싱(api-budget): 신경망 벡터는 (콘텐츠·모델) 해시로 npz 캐시(`neural_section_vectors.npz`) →
  재실행 재계산 0. 임베딩은 전부 로컬(유료 호출 0).
- ★ dev(2016~2022) 전용. holdout 미개봉.

## 결과 — 5축 그리드(L4 절차) 텍스트만 교체

| 텍스트 벡터라이저 | 학습 텍스트 가중치 | dev 표본 APE 중앙값 | 나머지 축 가중치(산업/규모/시총/성장) |
|---|--:|--:|---|
| **TF-IDF**(문자 3-gram, L4) | **0.2857** | **0.4838** | 0.214 / 0.286 / 0.143 / 0.071 |
| **신경망**(potion-multilingual) | **0.3636** ↑ | **0.4921** ↑(악화) | 0.182 / 0.273 / 0.091 / 0.091 |

※ TF-IDF 행은 L4 의 학습 결과(가중치·subsample APE 0.4838)를 **정확히 재현** — 정찰 하네스 정합성 확인.

## ★ 판정: 유료 임베딩 가치 = **낮음(LOW)**

두 관측이 **엇갈린다(코드 라벨 MIXED)** — 그러나 유료 지출 관점의 결론은 **낮음**이다:

1. **신경망 텍스트가 '더 신뢰받는다'(가중치 ↑): 0.286 → 0.364.** 그리드가 의미 임베딩에 더 큰
   가중치를 줬다 = 문자 n-gram(어휘 표면)보다 **의미 유사도가 in-sample 로 더 정보적**이다.
   → 신경망 임베딩이 사업내용의 의미를 실제로 더 잘 잡는다는 약한 긍정 신호.

2. **★ 그러나 정작 중요한 지표(APE)는 개선되지 않았다: 0.4838 → 0.4921(오히려 소폭 악화).**
   더 나은 표현(신경망)이 텍스트 축 가중치는 올렸지만 **peer-중앙값 예측 정확도를 낮추지 못했다.**
   높은 텍스트 가중치는 산업·시총 가중치를 잠식했고, 순효과는 **개선 0**.

**해석:** "표현을 더 정교화해도(전문→섹션→성장→부문→**신경망 의미 임베딩**) median APE 는
거의 안 움직인다"는 **4루프 연속 상한 관측을, 진짜 신경망 임베딩이 확증**한다. 텍스트 축의
**형태(lexical→semantic)를 바꿔도 peer 선정의 median 예측력은 상한(≈0.48~0.50)에 갇혀 있다.**
유료 최신 임베딩(더 큰 모델·문맥)은 이 무료 소형 정적 임베딩보다 표현이 정교할 수 있으나,
**무료 신경망조차 APE 를 못 낮췄다**는 사실은 유료의 기대 효용이 **낮음**임을 강하게 시사한다.
0.433(승리조건)을 임베딩이 깰 근거는 4루프째 나타나지 않았다.

## 한계 (정직히)
- 정적(static) 임베딩이라 문맥(context) 반영이 얕다 — 유료 문맥 임베딩(예: 대형 트랜스포머)은
  표현이 더 풍부할 수 있다. 그러나 (1) 이번 무료 신경망이 이미 TF-IDF 보다 **의미를 더 잡았고**
  (가중치 ↑), 그럼에도 APE 개선이 0인 점, (2) Loop 2~5 의 표현 축 전부가 median 상한에 수렴한 점을
  종합하면, 유료 지출로 median 이 유의미하게 내려갈 개연성은 낮다.
- subsample(250) 기반 정찰이다. 신경망의 텍스트 가중치 상승이 subsample 과적합일 수 있어(full-dev
  일반화 저하), '개선 없음'은 오히려 **보수적으로도 성립**한다(과적합이면 실제 효용은 더 낮음).
- 이 정찰은 "peer 선정용 텍스트 유사도"에 한정. 산출물의 다른 용도(개별 사업내용 검색·요약)에서
  신경망 임베딩의 가치는 별개다(이 채점 목표와 무관).
