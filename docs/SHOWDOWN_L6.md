## ★ C(예측불가 분리): 안정 APE + 제외 비율
- 구조적 플래그 s=|비율분자|/총자산 < τ_r(dev 하위 10%, D-027). **제외 9.9%(<20%).**
- **안정 APE(전체 대비 약 −0.035):** L6 0.4794→**0.4452**, L4 0.4977→0.4627, baseline 0.5427→0.5016.
  ill-conditioned(손익분기 근처) 9.9% 를 빼면 well-conditioned 성능이 드러난다. 상대비교 불변.
- 감사 활용: 이 9.9% 는 "동종업계 비교 부적합" 플래그(red-flag 아님, SEPARATION.md).

## ★ 신경망 임베딩 무료 정찰: 유료 가치 = **낮음(LOW)** (EMBEDDING_SCOUT2.md)

## ★ 승리조건(0.433) + 정직한 결론
- **미달** — 그러나 **처음으로 유의하게 전진.** 전체 0.4794 / 안정 0.4452, 둘 다 0.433 초과.
  단 gap 이 뚜렷이 좁아졌다(L4 0.4977→L6 0.4794, 안정 0.4452 는 0.433 에 근접, +2.8%).
- **벽을 넘었나?** — **부분적으로 넘었다.** Loop 2~5 의 "peer **선정** 축을 더해도 median 이 상한(≈0.4965)에
  갇힌다"는 벽은, **선정이 아니라 예측 head(k)** 라는 **미탐색 지렛대**로 **유의하게 돌파**됐다(−3.68%).
  ★ 단 정직히: 이 개선은 **더 똑똑한 선정·집계가 아니라 단지 peer 를 5→10 개** 쓴 것 — 앞선 루프들이
  ORACLE 기본 k=5 를 고정한 데서 온 **자가부과 벽**의 일부였다. 값싼 개선이지만 **실재하고 유의**하다.
- **남은 격차(0.4794→0.433, 약 −7.6%)** 는 여전히 크다. k 를 더 키우면 시장중앙값(0.5724)으로 U자
  회귀하므로 k 만으로는 못 메운다. 무료 표현축(부문·성장·신경망 텍스트)은 전부 median 상한에 수렴.

## ★ holdout 개봉 권고 (결정은 사람)
- **무료 카드 상태:** 선정 축(산업·규모·시총·텍스트·성장·부문) 소진(L5), **예측 head(k)** 는 이번에
  소진(k=10 near-optimal). 신경망 임베딩(무료 정찰) LOW. **남은 무료 지렛대는 사실상 없다.**
- **★ 이번 루프의 새 사실:** L6 가 L4 를 **유의하게** 이겼다(dev). 이것은 **holdout 개봉의 실질적 이유**를
  처음 만든다 — "L4 가 상한"이라던 3루프 관측이 **틀렸고**(예측 head 미탐색), L6 의 dev 우위(−3.68%)가
  **out-of-sample 에서도 유지되는지 검증할 값어치**가 생겼다.
- **권고:** **(1) holdout 개봉 검증**을 우선 고려. 근거: (a) 무료 지렛대 소진, (b) L6 가 dev 에서 처음
  유의 개선 → 검증 대상이 명확, (c) 임베딩(유료)은 무료 정찰상 LOW → 개봉 전 지출 불필요.
  개봉 시: L6(median@k10) 과 L4(median@k5) 를 holdout 에서 채점해 **k 개선의 out-of-sample 재현**과
  **상한(0.433 미달) 확정**을 함께 본다. 1회성 봉인해제이므로 사람이 결정.
